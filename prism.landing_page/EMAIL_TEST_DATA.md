## 🎯 Cách sử dụng Test Data

### 1. Test trong EmailJS Dashboard
1. Vào template của bạn
2. Click nút "Test It"
3. Copy paste từng field từ test data trên
4. Click "Send Test Email"
5. Check inbox

### 2. Test trên Website
1. Mở website đang chạy dev server
2. Scroll xuống Contact section
3. Fill form với test data
4. Click "Gửi liên hệ"
5. Đợi toast notification
6. Check email

### 3. Test với các Edge Cases
- Email rất dài
- Số điện thoại không điền
- Nội dung có special characters: @#$%^&*
- Nội dung có line breaks và formatting
- Subject là "Khác" vs options khác
